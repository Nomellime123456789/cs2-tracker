# \# CS2 Match Tracker 🎮

# 

# Персональный трекер статистики матчей в Counter-Strike 2.

# 

# \## Описание

# Программа позволяет вручную записывать результаты матчей CS2

# и анализировать свою игру с течением времени.

# 

# \## Возможности

# \- Добавление результатов матча (убийства, смерти, помощи, урон, карта)

# \- Подсчёт K/D, винрейта, среднего урона за раунд

# \- Статистика по картам

# \- Графики прогресса

# 

# \## Технологии

# \- Python 3

# \- pandas

# \- matplotlib

# 

# \## Как запустить

# 

# \### Установить зависимости

# pip install -r requirements.txt

# 

# \### Запустить программу

# python main.py

#

## Функционал

### Меню программы
- **1. Добавить матч** — ввод статистики после игры
- **2. Показать статистику** — K/D, винрейт, ADR, лучший/худший матч
- **3. Показать графики** — визуализация прогресса

### Пример статистики
```
=== 📊 Общая статистика ===
Всего матчей:       4
K/D ratio:          2.31
Винрейт:            100.0%
Средний ADR:        77.0
```
